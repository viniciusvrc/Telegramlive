# Telegramlive
TelegramLiveRelay é um aplicativo desktop feito em C# com WinForms que captura uma URL do YouTube e retransmite o conteúdo   para um servidor RTMP do Telegram. O programa usa yt-dlp para obter o stream e ffmpeg para processar, redimensionar,   ajustar o áudio e manter a transmissão estável.
